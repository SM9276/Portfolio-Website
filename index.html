My Computer Engineering Journey

2012

The first computer I interacted with was my father's Windows XP machine, which I was not supposed to touch. Since I was always so curious, I ended up dismantling it. I really wanted to know how this box displayed an image. My father was really mad that night!

2015

I got into installing custom Android OS, running LineageOS with Android 7.1.2. I helped my community by unlocking SIM cards and improving their phones for farm workers.

2016

A series of slow cheap laptops led me to want to build my own desktop computer. I still use this machine to this day; I've upgraded the GPU, but the i7-7700 is still going strong.

2018-2019

I took my first computer science class in my freshman year of high school. We learned the fundamentals of Boolean logic and did assignments in HTML and CSS.

2019-2020

Sophomore year of high school, I took AP Computer Science, where I learned about Object-Oriented Programming and built assignments in Java.
